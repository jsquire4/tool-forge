# Stage: Name & Describe

Goal: lock the tool's name, description, and trigger phrases.

Requirements:
- name: snake_case, no spaces, reflects what the tool does
- description: one sentence, precise, includes when to use AND when NOT to use (disambiguation)
- triggerPhrases: 3+ natural language variations a user might say to invoke this tool

Do not proceed until the user has confirmed all three. Show a summary before asking for confirmation.
